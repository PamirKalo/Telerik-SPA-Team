# Telerik-SPA-Team 4
Organization: Telerik Alpha 2018
This is a simple Single Page Application(SPA) project.
Purpose: education, work with JQuery, DOM manipulations with JavaScript, SPA structure, AJAX, project timing, Git-clients and GitHub, Trello - task managing sftware, team colaboration.
Kind: memory game.
Short description: one-player game. The player choose two cards. Should one open card match with the another, they both stay revealead. If not match they are hidden again. Purpose is to reveal all the card on the field...
Field or game board can be costumize by the player. He can choose from three sized game boards with 16, 20 and 24 cards. Moreover it is also possible to select three card types:
 - with words - JS terminology
 - with cats pictures
 - with colors
The game lasts five minutes.
