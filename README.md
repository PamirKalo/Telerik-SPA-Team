# Telerik-SPA-Teamproject - Team 4
__Organization:__ Telerik Alpha 2018  
This is a simple Single Page Application(SPA) project.  
**Purpose:** education, excercise JQuery, DOM manipulations with JavaScript, SPA structure, AJAX, project timing, Git-clients and GitHub, Trello - task managing software, team colaboration.  
**App Type:** Memory card game.  
**Short description:** single player game. The player choose two cards. Should the opened card match with another, they both stay revealed. If not, they are hidden again. Purpose is to reveal all the cards on the field...  
The field or game board can be customized by the player. He can choose from three game board sizes - 16, 20 or 24 cards. Moreover it is also possible to select three card types:  
 - with words - JS terminology
 - with cats pictures
 - with colors 
 
The game lasts five minutes.  

## __Game interface__: 
Leftsidebar is a leaderboard with player score and timing. Which is toggled, by the button on the top left corner.

 **Customization screen** - Game board size is adjusted using the slider, and the card types can be choosen by the radio buttons. The game board is created by the green "Create Board" button.  

**Game board screen** the actual game starts with 5 min countdown, after the button Start Game is pressed. The remainig time for the game, is visualized in a stripped progress bar below the timer. The current score is right below the timer of the game, and on it's right, the player has an input field, to enter his Name, if he wants to save his score to the Leaderboard. A Wiki Help button is above the gameboard, and its purpouse is to help players with JS Terminology they don't remember.
